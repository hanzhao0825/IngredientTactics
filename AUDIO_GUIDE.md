# 音频文件说明 (Audio Files Guide)

## 文件夹结构

请在项目根目录创建 `audio` 文件夹，并将你的音乐文件放入其中：

```
IngredientTactics/
├── audio/
│   └── bgm.mp3          # 背景音乐文件
├── img/
├── src/
└── index.html
```

## 支持的音频格式

- **推荐格式**: MP3 (最广泛支持)
- **备选格式**: OGG, WAV

## 如何添加你自己的 BGM

1. 创建 `audio` 文件夹（如果还不存在）
2. 将你的音乐文件命名为 `bgm.mp3`
3. 放入 `audio/` 文件夹
4. 刷新游戏页面

**如果文件加载失败，游戏会自动回退到程序化生成的 BGM。**

## 音乐文件建议

- **时长**: 1-3 分钟循环
- **音量**: 适中，不要太响
- **风格**: 轻松愉快的背景音乐
- **文件大小**: 建议 < 5MB

## 更换 BGM

如果你想使用不同的文件名，可以修改 `src/game.js` 中的这一行：

```javascript
this.audio.playBGM('audio/bgm.mp3');  // 改成你的文件名
```
